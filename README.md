# 🚗 Smart Rover V3

An ESP32-based Smart Rover featuring Wi-Fi control, computer vision-based face tracking, and ultrasonic obstacle avoidance.

---

## ✨ Features

- 📡 Wi-Fi Web Control
- 🚗 Dual DC Motor Drive
- 👤 Face Tracking using OpenCV + MediaPipe
- 📏 Ultrasonic Obstacle Avoidance (HC-SR04)
- 🎯 Servo-based Object Scanning
- 🌐 ESP32 Web Server
- ⚡ Real-time Wireless Control

---

## 🛠 Hardware

- ESP32 Dev Module
- L293 Motor Driver
- 2 × TT DC Motors
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Breadboard
- Power Bank
- Chassis Kit

---

## 💻 Software

- Arduino IDE
- Python 3
- OpenCV
- MediaPipe
- ESP32 Arduino Core

## 🚀 Installation

### Arduino

1. Install ESP32 Board Package
2. Open `Arduino/Smart_Rover_V3.ino`
3. Enter your Wi-Fi SSID & Password
4. Upload to ESP32

### Python

```bash
pip install -r requirements.txt
python face_tracking.py
```
